# sso

This is a demo application created to test SSO and get raw tokens to check all claims in jwt.io
- cd into SSOMinimalApiDemo Project before running

## Pre-requirements

- New to registes a application in any IdentityProvider(Azure) and configure endpoints for sign-in and sign-out
- Set-up claims to send back with the token
- Configure the tenant-Id and client-Id in appsetting.json which are provided by the registered application
